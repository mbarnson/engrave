  <action>
    **InferenceRouter (router.py):**
    - Per RESEARCH.md Pattern 1 -- `InferenceRouter` class taking `Settings` in constructor
    - `async def complete(role, messages, temperature=0.3, max_tokens=None) -> str`
    - Resolves role to RoleConfig, calls `litellm.acompletion()` with fully-qualified model string (provider prefix required per Pitfall 1)
    - Sets `num_retries=0` per user decision (fail, don't fallback)
    - Sets api_base from role config (None for cloud, URL for local/RunPod)
    - Sets api_key from role config (resolved from .env via settings)
    - Raises `ProviderError` on any LiteLLM exception with provider name, model, and original error
    - Raises `RoleNotFoundError` if role is not in config

    **Exceptions (exceptions.py):**
    - `ProviderError(Exception)` with provider, model, original_error fields
    - `RoleNotFoundError(Exception)` with role name and available roles

    **CLI (cli.py):**
    - Typer app with `name="engrave"`, `help="AI-powered music engraving pipeline"`
    - `check` command: takes provider name (or "all"), sends a trivial test completion, reports success/failure with Rich output
    - `version` command: prints package version
    - `compile` command: stub for now (will be wired in Plan 02) that prints "Not yet implemented" -- placeholder so the CLI structure is complete
    - Use lazy imports inside command functions (not at module level) to keep CLI startup fast

    **Tests (test_router.py):**
    - Test router resolves role to correct model string
    - Test router calls acompletion with correct parameters (mock acompletion)
    - Test router raises RoleNotFoundError for unknown role
    - Test router raises ProviderError when acompletion fails (mock raises exception)
    - Test router passes api_base for local providers, None for cloud
    - Use `pytest-asyncio` for async test functions
    - All tests mock LiteLLM -- never make real API calls

    Important: Use `@pytest.mark.asyncio` on async tests per Pitfall 7. Mock at `litellm.acompletion` level.
