       - `ScoreMetadata`: source_collection, source_path, chunk_index, bar_start, bar_end, chunk_type (Literal["single_instrument", "full_score"]), key_signature, time_signature, tempo, instrument, instrument_family, clef, ensemble_type, style, composer, note_density (float|None), dynamic_range, articulation_count (int), has_chord_symbols (bool), has_midi (bool), rich_description (str, default="") — empty placeholder for Music Flamingo enrichment per user decision

       - `RetrievalResult`: chunk (Chunk), distance (float)

       - `get_embedding_function(model_name: str) -> chromadb.EmbeddingFunction` — factory function that returns a SentenceTransformer-based ChromaDB embedding function for the given model name
       - Uses `chromadb.utils.embedding_functions.SentenceTransformerEmbeddingFunction` with the model_name parameter
       - Default model is `nomic-embed-text` per user decision (configurable, NOT a dual-model hybrid)
       - The abstraction makes it trivial to swap models for A/B testing: change engrave.toml, restart

    4. Extend `src/engrave/config/settings.py`:

       - Add `corpus: CorpusConfig = CorpusConfig()` to the Settings class

    5. Update `engrave.toml`:
       - Add `[corpus]` section with `embedding_model = "nomic-embed-text"`, `db_path = "data/corpus_db"`, `collection_name = "lilypond_phrases"`

    6. Create `src/engrave/corpus/store.py`:
       - `CorpusStore` class wrapping ChromaDB PersistentClient
       - `__init__(self, config: CorpusConfig)` — creates client and collection with configurable embedding function via get_embedding_function(config.embedding_model)
       - `add_chunks(self, chunks: list[Chunk]) -> int` — adds chunks to collection; document=description, metadata=ScoreMetadata.model_dump() with LilyPond source stored in metadata field `ly_source`; returns count added
       - `query(self, query: RetrievalQuery) -> list[RetrievalResult]` — builds where clause from optional filters ($and combinator), queries with embedding similarity, returns RetrievalResult list
       - `count(self) -> int` — returns collection document count
       - Collection metadata: `{"hnsw:space": "cosine"}`
