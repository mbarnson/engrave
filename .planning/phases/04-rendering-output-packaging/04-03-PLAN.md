**Create `src/engrave/rendering/packager.py`:**

**`RenderPipeline` class:**
```python
class RenderPipeline:
    def __init__(
        self,
        preset: BigBandPreset,
        compiler: LilyPondCompiler | None = None,
        score_timeout: int = 300,
        part_timeout: int = 60,
    ): ...
```

**`render(self, music_vars: dict[str, str], global_music: str, chord_symbols: str | None, song_title: str | None, output_dir: Path) -> RenderResult`:**
1. Create a temporary work directory (or use output_dir)
2. Apply `restate_dynamics` to each instrument's music content in `music_vars`
3. Call `generate_music_definitions(music_vars, global_music, chord_symbols)` -> write `music-definitions.ly`
4. Call `generate_conductor_score(preset, ...)` -> write `score.ly`
5. For each instrument in preset: call `generate_part(instrument, ...)` -> write `part-{slug}.ly`
6. Compile `score.ly` with `score_timeout` (300s) -- produces score.pdf + score.mid
7. Compile each `part-{slug}.ly` with `part_timeout` (60s) -- produces part PDFs
8. Handle compilation failures gracefully: log errors, continue with remaining files
9. Call `package_zip(work_dir, output_path)` to assemble the ZIP

**`package_zip(self, work_dir: Path, output_path: Path) -> Path`:**
- Use `zipfile.ZipFile` with `ZIP_DEFLATED`
- Add all .pdf, .ly, .mid files from work_dir with flat names (just filename, no directory)
- Return the output path

**`RenderResult` dataclass:**
- `zip_path: Path` -- path to output ZIP
- `success: bool` -- True if score and all parts compiled
- `compiled: list[str]` -- filenames that compiled successfully
- `failed: list[str]` -- filenames that failed compilation
- `errors: dict[str, str]` -- filename -> error message for failures

**Song title resolution:** Priority: (1) explicit `song_title` param, (2) "untitled". Slugification: lowercase, replace spaces/special chars with hyphens, strip leading/trailing hyphens. Simple helper function using `re.sub`.

**ZIP naming:** `{slugified_title}-{YYYY-MM-DD}.zip` using `datetime.date.today()`.
