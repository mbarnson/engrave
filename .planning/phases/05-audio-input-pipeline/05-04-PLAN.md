    1. Create `src/engrave/audio/pipeline.py` with:

       - `@dataclass class StemResult:` -- stem_name (str), midi_path (Path), wav_path (Path), quality (StemQuality), model_used (str)

       - `@dataclass class JobResult:` -- job_dir (Path), input_path (Path), stem_results (list[StemResult]), metadata (dict[str, Any]) containing pipeline config, timing, model versions

       - `class AudioPipeline:` with:
         - `__init__(self, config: AudioConfig, transcriber: Transcriber | None = None)` -- stores config, creates transcriber from config if not injected (supports testing)
         - `process(self, input_path: Path, job_dir: Path | None = None) -> JobResult:` -- orchestrates the full pipeline:
           a. Create job directory: `jobs/{timestamp}_{input_path.stem}/` if job_dir not provided
           b. **Normalize:** Call normalize_audio(input_path, job_dir / "input.wav", config.target_sample_rate, config.target_channels)
           c. **Separate:** Call run_separation(normalized_wav, config.separation.steps or get_default_steps(), job_dir / "separation")
           d. **Transcribe:** For each StemOutput from separation, call transcriber.transcribe(stem.path, job_dir / "transcription") -- returns MIDI path
           e. **Annotate:** For each transcribed MIDI, call annotate_quality(midi_path, stem.stem_name, tempo_bpm=estimated_tempo, expected_range=get_expected_range(stem.stem_name))
           f. Collect StemResult objects
           g. Write `job_dir / "metadata.json"` with pipeline config, timing, input path, stem count
           h. Return JobResult

         - `process_youtube(self, url: str, job_dir: Path | None = None) -> JobResult:` -- wraps extract_youtube_audio() then calls process()

       - Job directory structure (created by pipeline):
         ```
         jobs/{timestamp}_{name}/
         ├── input.wav
         ├── separation/
         │   ├── step-00_{model}/
         │   │   ├── drums.wav
         │   │   ├── bass.wav
         │   │   └── ...
         │   └── step-01_{model}/
         │       └── ...
         ├── transcription/
         │   ├── drums.mid
         │   ├── bass.mid
         │   └── ...
         ├── quality/
         │   └── stem_quality.json
         └── metadata.json
         ```

    2. Update `src/engrave/audio/__init__.py` to export the public API:
       - `from engrave.audio.pipeline import AudioPipeline, JobResult, StemResult`
       - `from engrave.audio.normalizer import normalize_audio`
       - `from engrave.audio.youtube import extract_youtube_audio`
       - `from engrave.audio.quality import StemQuality, annotate_quality`
       - `__all__` listing all exports

    3. Add to `tests/conftest.py`:
