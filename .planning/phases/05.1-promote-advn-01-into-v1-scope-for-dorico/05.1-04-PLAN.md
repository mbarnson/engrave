**test_musicxml_roundtrip.py:**
- Build JSON notation events with specific pitches, rhythms, and dynamics
- Call `assemble_musicxml()` to write MusicXML
- Re-read the MusicXML using `music21.converter.parse()`
- Verify note count matches
- Verify specific pitches survive roundtrip (e.g., B-flat remains B-flat, not A-sharp)
- Verify dynamics survive roundtrip (f, p, mf)
- Verify articulations survive roundtrip (staccato, accent)
- Verify measure numbers are correct
- Use `tmp_path` fixture

**test_musicxml_degradation.py:**
- Test 1: Pass `json_sections=[None, None]` (all sections failed JSON generation) to `assemble_musicxml()` -- assert returns (False, None), does not crash
- Test 2: Pass JSON with invalid Pydantic structure (missing required fields) -- assert returns (False, None), logs warning
- Test 3: Pass mixture of valid and None sections -- assert partial assembly works with available sections
- Test 4: Verify RenderPipeline with include_musicxml=True but json_sections=None produces ZIP without .musicxml but with .ly files
