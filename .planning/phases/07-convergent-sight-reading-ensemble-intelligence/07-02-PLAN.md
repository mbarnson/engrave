    - All 5 saxophones: `section_group="saxes"`

    - All 4 rhythm section instruments: `section_group=None` (ungrouped, keep per-instrument generation)

    **2. Create `src/engrave/generation/section_groups.py`:**

    Contains:

    a. `BeamStyle` enum with values `SWING = "swing"` and `STRAIGHT = "straight"`.

    b. `resolve_section_groups(instruments: tuple[InstrumentSpec, ...]) -> list[list[InstrumentSpec]]`:
       - Groups instruments by `section_group` field
       - Instruments with `section_group=None` each become their own single-element group
       - Returns list of groups sorted by the first instrument's `score_order` within each group
       - Preserves `score_order` within groups

    c. Beaming command constants and function:
       ```python
       SWING_BEAMING = (
           "  \\set Timing.beamExceptions = #'()\n"
           "  \\set Timing.baseMoment = #(ly:make-moment 1/4)\n"
           "  \\set Timing.beatStructure = 1,1,1,1\n"
       )

       STRAIGHT_BEAMING = (
           "  % Default LilyPond beaming (half-bar grouping in 4/4)\n"
           "  \\unset Timing.beamExceptions\n"
           "  \\unset Timing.baseMoment\n"
           "  \\unset Timing.beatStructure\n"
       )

       def beaming_commands(beam_style: BeamStyle | str) -> str:
           """Return LilyPond timing commands for the given style.
           Accepts BeamStyle enum or string 'swing'/'straight'.
           Defaults to swing if unrecognized."""
       ```

    d. `resolve_beam_style(section_description=None, user_hints=None) -> BeamStyle`:
       - Checks user hints first (authoritative), then audio description style field
       - Defaults to `BeamStyle.SWING` for big band if no signal
       - Latin, rock, pop -> STRAIGHT; swing, bebop, blues -> SWING

    **3. Update `tests/unit/test_ensemble.py`:**

    - Rhythm instruments each in their own group
    - Single-instrument preset returns one group per instrument

    **5. Create `tests/unit/test_beaming.py`:**
    Tests:
    - beaming_commands("swing") returns LilyPond 2.24 Timing properties with beamExceptions, baseMoment, beatStructure

    - Default beam_style is swing for big band
    - resolve_beam_style returns STRAIGHT for "latin", "rock" hints
    - resolve_beam_style returns SWING for "swing", no hints
