## Decisions Made
- Factory pattern (create_app) for the FastAPI app rather than module-level instance, enabling test isolation and cleaner imports
- asyncio.create_task for background pipeline execution instead of FastAPI BackgroundTasks, avoiding UploadFile closure (Pitfall 1) and async event loop issues (Pitfall 2)
- Task reference stored in jobs dict per ruff RUF006 guidance (prevent garbage collection of fire-and-forget tasks)
- TemplateResponse uses new request-first parameter order to comply with Starlette deprecation warning
- response_model=None on download endpoint to allow union return type (FileResponse | HTMLResponse)
